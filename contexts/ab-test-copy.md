# A/B Test Copy — Newo.ai New Segments
Prepared: April 20, 2026

---

## How to Use

Each section below has 3–5 headline/hook variants for a specific segment.
Test in order: run Variant A for 1 week, Variant B for 1 week, pick winner.
Use for: Google Ads headlines, landing page H1, LinkedIn post openers.

## A/B Test Tracking

Set up UTM parameters for each variant:
- `?utm_source=[channel]&utm_medium=[type]&utm_campaign=ab-test-[date]&utm_content=[variant]`

Track: CTR, signup rate, time-on-page. Declare winner after 500+ visitors or 2 weeks.
